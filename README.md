# TPS-NecroDancerLike
## NecroDancerLike game

一个仿照节奏地牢的第三人称游戏。
目前完成了角色的控制和4个技能的释放，和简单的敌人AI。
玩家的每一次行动需要根据背景音乐的节拍来进行操作。错过节拍则该次输入无效。
